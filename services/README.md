# Services

Servicos centrais ativos do JARVIS:

- `orchestrator-service`
- `memory-service`
- `governance-service`
- `operational-service`
- `knowledge-service`
- `observability-service`

Leitura correta:

- esses servicos compoem o baseline encerrado do `v1`;
- no ciclo atual, os mais sensiveis para implementacao sao `orchestrator-service`, `memory-service` e `observability-service`, por causa da trilha de continuidade entre missoes.

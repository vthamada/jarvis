# Shared

Area compartilhada para artefatos semanticos e tecnicos reutilizados por todo o sistema.

Ela concentra:

- contratos;
- tipos;
- schemas;
- eventos;
- estado;
- principios e identidade compartilhada.

Leitura correta:

- o baseline do `v1` ja depende dessa camada como base canonica;
- o `pos-v1` deve expandi-la com disciplina, sem quebrar APIs publicas do baseline sem necessidade real.

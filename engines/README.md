# Engines

Engines centrais ativas do JARVIS:

- `identity-engine`
- `executive-engine`
- `cognitive-engine`
- `planning-engine`
- `synthesis-engine`

Leitura correta:

- elas sustentam o baseline cognitivo encerrado do `v1`;
- no ciclo atual, `planning-engine` e `synthesis-engine` sao especialmente sensiveis para aprofundar continuidade entre missoes.
